# Buddy House

## Overview

Buddy House is the main virtual home of the Roblox Buddy AI.

It acts as the central hub where users interact with their avatar, unlock new systems, collect rewards, and access every major feature of the website.

## Purpose

- Main Home Screen
- Feature Navigation
- Daily Activities
- Progress Tracking
- Hidden Secrets
- Event Access
