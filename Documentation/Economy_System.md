# Economy System

## Overview

The Economy System powers all progression, rewards, shopping, seasonal content, and premium support within Roblox Buddy AI.

The system is designed to remain fair for every player while helping support long-term development of the project.

The game will never become Pay-to-Win.

---

# Main Currencies

The game contains three main currencies.

---

## Coins

Coins are the primary free currency.

Players earn Coins by:

- Story Horror Mode
- Hidden Ghost Mode
- Mini Games
- Daily Login
- Daily Missions
- Achievements
- Buddy Gifts
- Seasonal Events
- Exploration
- Hidden Collectibles

Coins are used for:

- Buddy Clothes
- Buddy House Decorations
- Furniture
- Normal Themes
- Basic Cosmetics
- General Shop Items

---

## Event Tokens

Each major event introduces its own Event Token.

Examples:

- Halloween Tokens
- Christmas Tokens
- Easter Tokens
- Anniversary Tokens

Event Tokens are earned only during their respective events.

They can be spent on:

- Event Outfits
- Event Decorations
- Event Mini Games
- Event Cosmetics
- Limited-Time Collectibles

Once unlocked, event items remain permanently owned.

---

## Buddy Gems

Buddy Gems are the Premium Currency.

Players may purchase Buddy Gems using real money.

Examples:

India Pricing (Example)

₹10 → 50 Buddy Gems

₹100 → 200 Buddy Gems

Pricing automatically adjusts depending on the player's country and local currency.

Buddy Gems can be used for:

- Premium Buddy Outfits
- Premium Themes
- Premium Decorations
- Exclusive Cosmetic Items
- Future Cosmetic Content

Buddy Gems never provide gameplay advantages.

---

# No Pay-to-Win Policy

Premium purchases never increase gameplay power.

Premium players will NOT receive:

- Stronger abilities
- Faster progression
- Story advantages
- Hidden Ghost advantages
- Event advantages

Premium content is cosmetic only.

---

# Daily Rewards

Players receive rewards for logging in.

Rewards include:

- Coins
- Event Tokens (during events)
- Small Bonuses
- Login Streak Rewards

Long login streaks provide additional rewards.

---

# Achievement Rewards

Completing achievements grants rewards.

Possible rewards include:

- Coins
- Event Tokens
- Special Cosmetics
- Rare Badges
- Buddy Gems (selected achievements only)

---

# Buddy Shop Economy

Players spend Coins and Buddy Gems inside Buddy Shop.

Buddy can also purchase items using Buddy's own savings.

Sometimes Buddy may ask the player for help if Buddy does not have enough Coins.

Helping Buddy increases friendship.

---

# Friendship Rewards

Helping Buddy may result in:

- Better Gifts
- Earlier Gifts
- Random Surprises
- Stronger Friendship Progression

Buddy remembers acts of kindness.

---

# Refund System

Players can refund accidental purchases.

Rules:

- Refund only within a limited time.
- Item must not have been used.
- Some limited or event items may not be refundable.

This system helps prevent accidental purchases.

---

# Seasonal Sales

Special sales occur during major events.

Examples:

- Halloween Sale
- Christmas Sale
- Anniversary Sale
- Summer Sale

Selected cosmetics and decorations may become temporarily discounted.

---

# Supporting Development

Purchasing Buddy Gems helps support:

- Better Servers
- AI Improvements
- Faster Updates
- New Features
- Future Events
- Long-Term Development

---

# Supporter Recognition

Players who support the project may receive special profile titles.

Examples:

- Supporter
- Bronze Supporter
- Silver Supporter
- Gold Supporter
- Diamond Supporter

Supporter titles are recognition only.

They never affect gameplay.

---

# Long-Term Goal

The Economy System is designed to:

- Reward active players.
- Keep progression fair.
- Support long-term development.
- Never become Pay-to-Win.
- Allow both free and premium players to enjoy the complete Roblox Buddy AI experience.
