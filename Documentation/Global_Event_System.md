Global_Event_System.md 🌍 Global Event System Overview 

The Global Event System is the foundation for all seasonal and worldwide events in Roblox Buddy AI.

Examples:

🎃 Halloween 🎄 Christmas ❤️ Valentine's Week ☀️ Summer Festival ❄️ Winter Festival 🌸 Spring Festival 🍂 Autumn Festival 🌍 Earth Day 🤝 Friendship Festival ❤️ Roblox Buddy AI Anniversary 

Each event follows the same system while having unique themes, rewards, stories, collectibles, and activities.

📅 Automatic Event Timeline 

Every event has:

Start Date End Date Daily Unlock System Automatic Timeline Progression 

No manual activation is required.

Example:

Day 1 → Automatically unlocks

Day 2 → Unlocks after Daily Reset

Day 3 → Unlocks after Daily Reset

...

Final Day → Automatically unlocks

🕓 Daily Reset 

Daily Reset Time

4:00 AM (Server Time)

At every reset the system automatically:

Unlocks the next Event Day Unlocks new missions Unlocks new shop items Unlocks new Buddy dialogue Unlocks new Story chapters Unlocks new Mini Games Unlocks new achievements Sends Event Mail Updates Event Roadmap 

No manual work is required.

Current Day

Countdown

🤖 Buddy Guide 

Buddy becomes the event guide.

Examples:

"Tomorrow Ghost Hunt unlocks!"

"Only 3 days until Secret Area!"

"Your next reward is almost ready!"

Buddy continuously helps the player throughout the event.

Day 2

🌍 Community Goal 

Entire community works together.

Example:

Collect 10,000,000 Pumpkins

After completion everyone receives rewards.

Players with higher contribution receive bonus rewards.

🏆 Event Leaderboard 

Every Global Event has three leaderboards.

🌍 Global Leaderboard 

Top 200 players worldwide.

🌏 Regional Leaderboard 

Top 200 players from the player's country.

👬 Friends Leaderboard 

Shows rankings among the player's friends only.

Used to compare progress with friends.

👑 Live Crown 

During the event:

Current Global Rank #1 receives a Live Crown.

If another player becomes Rank #1

The crown automatically transfers.

⏳ Event Ending 

When the event officially ends:

Story closes

Shop closes

Mini Games close

Hidden Areas close

Horror Mode closes

Players receive an additional 24-hour Reward Claim Period before the event completely finishes.

Long-Term Goal 

The Global Event System serves as the universal framework for every seasonal celebration in Roblox Buddy AI, providing automatic event progression, dynamic content unlocking, community participation, permanent champion history, museums, memory albums, and long-term collectible progression.
